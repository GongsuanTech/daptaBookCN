Glossary

Glossary#

Component#

Components are defined as calculation blocks that appear as boxes in the workspace.

Connection#

A Connection is defined as a data link from an origin Component output handle (right-hand side handle) to a target Component input handle (left-hand side handle).

Run#

A single instance of the a session model analysis, with all its inputs and outputs, that involves one or more valid Components.
